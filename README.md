# Data Storage & Analytics Solution

Academic project completed as part of MSc Data Analytics.

## Technologies Used

- SQL Server
- SSIS
- SSRS
- Tableau
- Neo4j
- SQL

## Project Overview

Designed and implemented a data warehouse solution using the Iowa Liquor Sales dataset.

Key activities included:
- Dimensional modelling using a star schema
- ETL pipeline development using SSIS
- Data warehouse implementation in SQL Server
- Operational and management reporting using SSRS
- Interactive Tableau dashboards
- Graph database modelling and analysis using Neo4j

## Key Deliverables

- Sales Summary Report
- Vendor Analysis Report
- Monthly Sales Report
- County-wise Sales Analysis
- Interactive Tableau Dashboard
- Neo4j Graph Database Queries
